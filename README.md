# Lexigraph

一个简约、离线优先的考研英语词汇复习应用。

Lexigraph 将单词视为节点，将拼写相似、语义易混和真实误认视为连接。它不仅安排“什么时候复习”，也尝试理解“哪些词容易被你混在一起”。

## 当前功能

- 可安装并离线使用的 PWA
- 无账号、无服务器的本地复习记录
- 四级掌握反馈与间隔调度
- 基于编辑距离和误认记录的易混词图谱
- JSON 学习记录导入、导出
- 移动端界面、测试和 GitHub Pages 自动部署

当前只带用于验证功能的演示词库。完整考研大纲词库将在确认公开数据来源与许可后，通过可复现的数据管线导入。

## 运行

```bash
npm install
npm run dev
```

```bash
npm test
npm run build
```

## 数据与隐私

学习记录只保存在浏览器 localStorage，不会上传。应用代码使用 MIT License；词库数据将单独记录来源和许可。
